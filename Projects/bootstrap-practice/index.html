Responsive Flex Classes

All flex classes comes with additional responsive classes, which makes it easy to set a specific flex class on a specific screen size. The * symbol can be replaced with sm, md, lg, xl or xxl, which represents small, medium, large, xlarge and xxlarge screens.

Resize the browser window to see the effect.

d-flex
d-sm-flex d-md-flex d-lg-flex d-xl-flex d-xxl-flex
